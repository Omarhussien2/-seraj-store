## ملاحظات للباك إند (مرحلة لاحقة)

> هذا القسم مرجع للمستقبل — لما نبدأ نحوّل المشروع لـ Next.js + MongoDB

### القسم 1: مقالات — Data Schema

```json
// articles.json → لاحقاً: MongoDB collection "articles"
{
  "id": "art_001",
  "title": "إزاي تتعاملي مع العند عند طفلك؟",
  "excerpt": "نصايح عملية من أمهات جربوا...",
  "body": "... المقال كامل (HTML أو Markdown) ...",
  "category": "تربية",           // ["تربية", "صحة", "تعليم", "سلوك", "نفسية"]
  "image": "/uploads/articles/art_001.jpg",
  "videoUrl": "https://youtube.com/...",  // اختياري
  "readingTime": 5,              // بالدقائق
  "tags": ["عند", "سلوك", "٣-٦ سنين"],
  "publishedAt": "2026-04-01",
  "author": "فريق سراج"
}
```

**الباك إند المطلوب:**
- `GET /api/articles` — قائمة المقالات (مع pagination + filter by category + search)
- `GET /api/articles/:id` — مقال واحد كامل
- `POST /api/articles` — إضافة مقال (Admin only)
- **Admin Panel**: واجهة لإدخال المقالات بـ rich text editor + رفع صور
- **Search**: Full-text search بالعربي (MongoDB Atlas Search أو Algolia)

### القسم 2: فسحة حلوة — Data Schema

```json
// outings.json → لاحقاً: MongoDB collection "outings"
{
  "id": "out_001",
  "name": "حديقة الأزهر",
  "description": "حديقة تاريخية جميلة مناسبة للعائلات...",
  "image": "/uploads/outings/out_001.jpg",
  "city": "القاهرة",
  "area": "الدرب الأحمر",
  "coordinates": { "lat": 30.0396, "lng": 31.2625 },
  "priceRange": { "min": 20, "max": 50 },  // بالجنيه، 0 = مجاني
  "ageRange": { "min": 3, "max": 10 },
  "category": "حدائق",          // ["حدائق", "متاحف", "ملاهي", "مطاعم", "أنشطة"]
  "tags": ["مناسب للأطفال", "مفتوح", "عائلي"],
  "rating": 4.5,
  "openingHours": "٩ ص — ١٠ م",
  "phone": "02-XXXXXXX",
  "googleMapsUrl": "https://maps.google.com/..."
}
```

**الباك إند المطلوب:**
- `GET /api/outings` — قائمة الأماكن (filter by city, area, budget, age)
- `GET /api/outings/:id` — مكان واحد بالتفاصيل
- `POST /api/outings` — إضافة مكان (Admin only)
- **Geolocation**: استخدام browser geolocation API لتحديد موقع الأم + ترتيب النتائج بالقرب
- **Budget Filter Logic**: `priceRange.max <= selectedBudget`
- **Admin Panel**: واجهة إدخال الأماكن مع خريطة لتحديد الموقع

### القسم 3: اسألي زينب (شات بوت) — Integration Plan

```
Architecture:
┌─────────────┐     ┌──────────────┐     ┌─────────────────┐
│  Frontend    │────▶│  Next.js API │────▶│  Claude API     │
│  Chat UI     │◀────│  /api/chat   │◀────│  (Anthropic)    │
└─────────────┘     └──────────────┘     └─────────────────┘
```

**الباك إند المطلوب:**
- `POST /api/chat` — إرسال رسالة + استقبال رد زينب
- **System Prompt**: شخصية الجدة زينب (أم مصرية حكيمة، عامية مصرية، نصايح تربوية)
- **Claude API** (Anthropic): الـ LLM اللي هيشغل زينب
- **Rate Limiting**: حد أقصى رسائل لكل مستخدم (مثلاً 20 رسالة/يوم)
- **Chat History**: حفظ المحادثات في MongoDB (اختياري)
- **Safety**: فلترة المحتوى + التأكد إن الردود مناسبة ومتخصصة في التربية

**System Prompt مقترح لزينب:**
```
أنتِ الجدة زينب، حارسة الحكايات في عائلة سراج.
أنتِ أم مصرية حكيمة بتتكلمي بالعامية المصرية.
بتساعدي الأمهات في تحديات التربية بحب وحنية.
ردودك قصيرة وعملية ومن القلب.
لو السؤال مش متعلق بالتربية أو الأطفال، وجّهي الأم بلطف.
```

## التحقق
1. فتح `index.html` في المتصفح
2. التأكد من ظهور "عالم ماما" في المنيو (top + bottom + footer)
3. الضغط على "عالم ماما" → الصفحة تظهر بالهيرو والتابات
4. التبديل بين التابات الـ 3 → كل قسم يظهر مع "قريباً"
5. فتح صفحة الويزارد → الأكورديون يظهر قبل Step 1
6. فتح/قفل الأكورديون يعمل بشكل صحيح
7. صفحة "حكايتنا" → اسم الجدة "زينب" مش "فاطمة"
8. تجربة على mobile (320px) — البوتوم ناف + التابات تشتغل كويس
